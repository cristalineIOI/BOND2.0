# BOND 2.0

Clone local du site marketing [bondapp.io](https://www.bondapp.io/) + page produit **FREDO** (agent vocal).

Repo : [github.com/cristalineIOI/BOND2.0](https://github.com/cristalineIOI/BOND2.0)

## Lancer

```bash
npm start
```

Ouvre http://localhost:3000

- Accueil : `/`
- FREDO : `/fredo/`

## Notes

Les assets Framer (images / fonts / modules) restent chargés depuis le CDN `framerusercontent.com` — une connexion internet est nécessaire pour l’affichage complet des pages clonées.
